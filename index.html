Lennox model number & capacity
Find the 3-digit BTU code embedded in the model number — it appears after the series prefix letters.
BTU code × 1,000 ÷ 12,000 = tons  ·  e.g. ZGB150S4BM1G → 150×1000÷12000 = 12.5 tons
Common codes: 024=2T · 030=2.5T · 036=3T · 042=3.5T · 048=4T · 060=5T · 072=6T · 090=7.5T · 120=10T · 150=12.5T
Example: KGB060S4DU4G → 5 tons  ·  ZGB060S4BH1G → 5 tons
NEW — CX series: CX models use positions 5–6 ÷ 12. Example: CX35-60S-1H → 60/12 = 5 tons.

York model number & capacity
Format: XXX NNN ··· — 2–3 letters then a 2–3 digit BTU code
BTU code × 1,000 ÷ 12,000 = tons  ·  e.g. YHE24B21SA → 2 tons
Common codes: 18=1.5T · 24=2T · 30=2.5T · 36=3T · 42=3.5T · 48=4T · 60=5T
Trane / American Standard / Ameristar / Oxbox / GE — model & capacity
Residential split systems — the 3-digit BTU code appears after the brand/series prefix (typically around positions 7–9).
BTU code × 1,000 ÷ 12,000 = tons
Examples: 4A7A4024L1000A → 2T · 4A7A4036B1000A → 3T · 4A6H5048H1000A → 4T
Light commercial condensing units (TTA/TWA) — 3-digit code after prefix = nominal MBh:
090=7.5T · 120=10T · 150=12.5T · 180=15T · 240=20T · 300=25T
Commercial packaged (YHC/THC) — digits 4–6 = nominal MBh: 037=3T · 047=4T · 067=5T · etc.
Large commercial (GAC/EAC) — digits 4–6 = nominal MBh: 180=15T · 210=17.5T · 240=20T · 300=25T
Common codes: 018=1.5T · 024=2T · 030=2.5T · 036=3T · 042=3.5T · 048=4T · 060=5T · 072=6T · 090=7.5T · 120=10T · 150=12.5T · 180=15T
NEW — Trane "S?H?" commercial (pos 1=S · pos 2=A/E/F/L/S/X · pos 3=H · pos 4=L/K): tons read DIRECTLY at positions 6–7. Most codes = direct tons (20,25,30,40,50,55,60,70,75,24,29,36,48,59,73,80,89,90). Special: 11=105T · 12=115T · 13=30T.
Trane, American Standard, Ameristar, Oxbox and GE residential/commercial HVAC all share this naming structure. The 3-digit BTU/MBh code is always the key.
Rheem / Ruud — model number & capacity
Find the first 2–3 digit numeric group that follows a letter in the model number.
That code × 1,000 ÷ 12,000 = tons  ·  e.g. RGEDZT120ACG → 10 tons  ·  RGEDZR090ACB → 7.5 tons
Common codes: 018=1.5T · 024=2T · 030=2.5T · 036=3T · 042=3.5T · 048=4T · 060=5T · 072=6T · 090=7.5T · 120=10T · 150=12.5T · 180=15T
Split-system condensers (UA/RA series): e.g. UA16AZ24AJ3 → 24÷12 = 2T · UA16AZ48AJ3 → 4T
Rheem and Ruud are sister brands — identical serial and model structure.
Coleman — model number & capacity
Coleman (York UPG) uses the same BTU code logic as York — find the 2–3 digit code in the model.
Code × 1,000 ÷ 12,000 = tons  ·  Common: 018=1.5T · 024=2T · 036=3T · 048=4T · 060=5T · 072=6T · 090=7.5T
How to read a Coleman serial number — 4 styles
Style 1 (Oct 2004+): [Plant][Year digit][Month letter][Year digit][sequence]
Year = pos[1] + pos[3] combined  ·  Month = pos[2] letter
Example: W0K5896070 → Year=05=2005 · K=October → October 2005
Style 2 (1971–Oct 2004): [Plant][Month letter][Year letter][Type letter][sequence]
Year = pos[2] letter (table)  ·  Month = pos[1] letter (A=Jan…M=Dec)
Example: WAKM011379 → A=January · K=1980 or 2001 → January 1980/2001
Year letters: A=1971/92 · B=72/93 · C=73/94 · D=74/95 · E=75/96 · F=76/97 · G=77/98 · H=78/99 · J=79/2000 · K=80/2001 · L=81/2002 · M=82/2003 · N=83/2004
Style 3 (1992+, all digits): [YY][MM][5-digit seq]
First 2 digits = year ≥ 92  ·  next 2 = month (01–12)
Example: 990697368 → Year=1999 · Month=06=June → June 1999
Style 4 (pre-1992, all digits): [MM][YY][6-digit seq]
First 2 digits = month (01–12)  ·  next 2 = year
Example: 0188123456 → Month=01=January · Year=1988 → January 1988
Style 2 years may be ambiguous (two possible decades) — use equipment condition to confirm. Style 1 began October 2004 under York UPG manufacturing.
Mitsubishi — model number & capacity
Find the 2–3 digit capacity code (BTU/h × 1,000) in the model — it follows the family/type/zone prefix.
Code × 1,000 ÷ 12,000 = tons  ·  e.g. MXZ-4C36NAHZ2-U → 3T · PUZ-A24NHA7-BS → 2T
Common codes: 09=0.75T · 12=1T · 15=1.25T · 18=1.5T · 24=2T · 30=2.5T · 36=3T · 42=3.5T · 48=4T · 54=4.5T · 60=5T
Mini-split family prefix: M=Mini-split · S=Mini-split  ·  Professional prefix: P=Professional
How to read a Mitsubishi serial number
Style 1: 1st digit = last digit of year (decade must be assumed) · 2nd character = month
Month encoding: 1–9 = Jan–Sep  ·  X = Oct  ·  Y = Nov  ·  Z = Dec
Example: 89D00488D → Year=8 → 2008 or 2018 · Month=9=September
Example: 7003261T → Year=7 → 2007 or 2017 · Month=0 → unclear (check label for MFR date)
The decade cannot be determined from the serial alone — use equipment condition, refrigerant type (R-22 pre-2010, R-410A post-2010), or the MFR date on the data plate to confirm.
Goodman — model number & capacity
Find the 2–3 digit BTU code (×1,000) in the model — always divisible by 6.
Code × 1,000 ÷ 12,000 = tons  ·  e.g. GSXB403610 → 3T · CAPF4860C6DB → 60 → 5T
From Goodman nomenclature (pos 7–8 in split-system models): 18=1.5T · 24=2T · 30=2.5T · 36=3T · 42=3.5T · 48=4T · 60=5T
Note: In air handler models (CAPF, ARUF…) the code may appear embedded — look for 2–3 digits divisible by 6 or 12.
How to read a Goodman serial number
Style 1 (all digits): [YY][MM][sequence]
Pos 1–2 = 2-digit year · Pos 3–4 = 2-digit month (01–12)
Example: 9704011000 → Year=1997 · Month=04=April → April 1997
Style 2 (PTAC units only — ends in P or D): [Factory][Month letter][Year letter][sequence][P or D]
Pos 1 = factory code (ignored) · Pos 2 = month letter · Pos 3 = year letter
Month letters: A=Jan · B=Feb · C=Mar · D=Apr · E=May · F=Jun · G=Jul · H=Aug · J=Sep · K=Oct · L=Nov · M=Dec
Year letters: A=1990 · B=91 · C=92 · D=93 · E=94 · F=95 · G=96 · H=97 · J=98 · K=99 · L=2000 · M=01 · N=02 · P=03 · Q=04 · R=05 · S=06 · T=07 · U=08 · V=09 · W=10 · X=11 · Y=12 · Z=13
Example: 5FU5472618P → F=June · U=2008 → June 2008
Style 2 was used only on PTAC units made by Daikin for Goodman — always ends in the letter P or D. Style 1 is the standard format for all other Goodman equipment.

Serial Number (optional — for manufacture date)

Found on the unit's data label / rating plate

How to read a Lennox serial number (Style 1 — 1973+)
Format: PP YY M NNNNN — 2 plant digits + 2 year digits + 1 month letter + sequence
Positions 3–4 = Year  ·  Position 5 = Month letter (A–M)
Month letters: A=Jan · B=Feb · C=Mar · D=Apr · E=May · F=Jun · G=Jul · H=Aug · J=Sep · K=Oct · L=Nov · M=Dec
Example: 1606B13871 → Year 2006 · B = February → February 2006
Example: 5899L17212 → Year 1999 · L = November → November 1999
NEW — Strict 10-11 digit validation: Canonical Lennox serials have 10 or 11 characters. When the length matches AND the year is ≤ current year, the result is flagged as Strict 10-11 digit format (highest confidence). Shorter serials still work for backward compatibility.
Style 1 has been used exclusively since 1973. First two digits = factory/plant code (ignored for date).
How to read a Carrier serial number
Format: WW YY L NNNNN — must start with 4 digits + a letter
Positions 1–2 = Week  ·  Positions 3–4 = Year  ·  Position 5 = Letter (ignored)
Example: 40 06 A 17330 → Week 40 of 2006 → October 2006
Example: 19 23 C 08821 → Week 19 of 2023 → May 2023
Carrier has used this format exclusively since 1980 · Week → Month conversion is approximate (±2 weeks)
How to read a York serial number
Style B (2004+): P D M D ······ — Plant · Year digit · Month letter · Year digit · seq
Example: W0K5896 → Year=05=2005 · K=September → Sep 2005
Style A (pre-2004): P M Y Y ······ — Plant · Month letter · Year letter · Year letter · seq
Example: WAKM011379 → A=January · KM → year 1982 or 2003
Month letters (both styles): A=Jan B=Feb C=Mar D=Apr E=May F=Jun G=Jul H=Aug K=Sep L=Oct M=Nov N=Dec
Style A years may be ambiguous (two possible values) — use equipment age/records to confirm
How to read a Trane / American Standard / Ameristar / Oxbox / GE serial number
Style 2 (pre-2002): [Factory letter] [YY] [Month letter] [sequence]
Pos 1 = factory code (letter, ignored) · Pos 2–3 = 2-digit year · Pos 4 = month letter
Month letters: A=Jan · B=Feb · C=Mar · D=Apr · E=May · F=Jun · G=Jul · H=Aug · J=Sep · K=Oct · L=Nov · M=Dec
Example: A92M07217 → Year 1992 · M=December → December 1992
Style 1 (2002–2009): 1st digit = last digit of year · 2nd–3rd digits = fiscal week
Example: 91531S41F → Year 2009 · Week 15 → ~April 2009
Style 1 (2010+): 1st–2nd digits = 2-digit year · 3rd–4th digits = fiscal week
Example: 10161KEDAA → Year 2010 · Week 16 → ~April 2010
Example: 130313596L → Year 2013 · Week 03 → ~January 2013
Fiscal week → Month: 1–4=Jan · 5–8=Feb · 9–13=Mar · 14–17=Apr · 18–22=May · 23–26=Jun · 27–30=Jul · 31–35=Aug · 36–39=Sep · 40–43=Oct · 44–48=Nov · 49–53=Dec
NEW — Year-letter prefix (priority rule): If pos 1 is a year letter (W=1983, X=1984, Y=1985, S=1986, B=1987, C=1988, D=1989, E=1990, F=1991, G=1992, H=1993, J=1994, K=1995, L=1996, M=1997, N=1998, P=1999, R=2000, Z=2001) AND pos 2 (or 2–3) is the fiscal week, this format is used regardless of total serial length.
Example: R24103967D → R=2000 · Week 24 → ~June 2000 (works on 10-char serials too).
NEW — Invalid-week recovery for Style 1 (2010+): If digits 3–4 form a week > 53 (impossible), reinterpret as 1-digit-year + 2-digit-week.
Example: 136110672D → "13"+"61" invalid → recover as 1+"36" → 2001 · Week 36 → ~September 2001
Trane, American Standard, Ameristar, Ingersoll Rand, Oxbox and GE all share these formats. The MFR date is also commonly printed on the data plate.
How to read a Rheem / Ruud serial number — 6 styles
Style 1: [PlantLetter][WW][YY][seq]  ·  pos 2–3=week · pos 4–5=year
Example: W291013412 → Week 29 · Year 2010 → ~July 2010
Styles 2/3/4: digits·[PlantLetter][WW][YY]·seq — find the factory letter (F,M,G,W,N) in the middle; 4 digits after it: first 2=week, last 2=year
Style 2 example: CB5D302F099903346 → F is plant · Week 09 · Year 1999 → ~March 1999
Style 3 example: 7351 M2806 16735 → M is plant · Week 28 · Year 2006 → ~July 2006
Style 4 example: OR177 M2807 00097 → M is plant · Week 28 · Year 2007 → ~July 2007
Style 5: [MM][YY][PlantLetter][seq] — pos 1–2=month · pos 3–4=year · pos 5=plant letter
Example: 0804B10488 → Month 08=August · Year 2004 → August 2004
Style 6 (pre-1975, discontinued): [seq][WW][YY] — last 4 digits: 3rd+4th from end=week, last 2=year
Example: 123451872 → Week 18 · Year 1972 → ~May 1972
Plant letters: F=Fort Smith AR · M=Milligeville GA · G=Greenville MS · W=Wichita KS · N=Nuevo Laredo Mexico. Style 6 is discontinued (mid-1970s and prior).
